Logical Modelling Tool

The plan is to generate these pages from Markdown, with the following features:

  1. Each div marked with class="lmt" gets replaced by the logical modelling tool widget with the initial data.
  2. The data for each one is persisted using the browser local storage, keyed by the id of the element.
  3. Multiple divs can be linked together in sequence, with the output of the previous feeding into the later ones.

An example instance

Here is an example instance of the Logical Modelling Tool:

atom a atom b atom c ifsat(a | b | c) [a,b,c]

Some more explanatory text.

allsat((a | b | c) & ~a) [a,b,c]
